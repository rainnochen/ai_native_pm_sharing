# Builder Starters

Builder Starters help PMs turn product thinking into runnable evidence.

| Starter | Use |
|---|---|
| [`spec-to-demo/`](spec-to-demo/README.md) | Turn a problem, spec, and acceptance criteria into a demo plan |
| [`html-prototype/`](html-prototype/README.md) | Build a lightweight HTML prototype for review |
| [`agent-demo/`](agent-demo/README.md) | Design an AI Agent demo with tools, state, eval, and fallback |
| [`eval-driven-demo/`](eval-driven-demo/README.md) | Build demos around explicit test cases and failure modes |

Builder does not mean PM replaces engineers. It means PM can create concrete evidence before asking the organization to commit.
