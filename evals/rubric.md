# PM AI Output Rubric

| Dimension | Good Looks Like | Red Flag |
|---|---|---|
| Decision relevance | Supports a specific decision | Generic summary |
| Evidence | Claims tied to sources or inputs | Invented certainty |
| Completeness | Covers main path and edge cases | Only happy path |
| Safety | Human gates are clear | Risky action hidden |
| Actionability | Next step is concrete | Vague recommendation |

## Use

Score each dimension 1-5. Outputs below 3 on evidence or safety should not be used for decisions.
