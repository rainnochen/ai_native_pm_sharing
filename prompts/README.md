# Prompts

Prompts in this directory are designed as entry points into a broader PM Harness. They should not replace context, skills, human gates, or evaluation.

| Prompt | Use |
|---|---|
| [`ask-before-answer.md`](ask-before-answer.md) | Make the model ask clarifying questions before producing a PRD, analysis, or plan |
| [`course-development-loop-prompt.md`](course-development-loop-prompt.md) | Run the course development loop |

Rule: if missing information would change the decision, the model should ask first instead of inventing.
