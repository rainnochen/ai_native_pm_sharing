# Hackathon Repo Template

Use this structure for a hackathon project README.

```markdown
# Project Name

## Problem

## User

## Demo

## AI-Native Workflow

## Architecture

## How To Run

## Eval Cases

## Risks And Limitations

## Decision Log

## Team
```

## Quality Checks

- README includes a runnable command.
- Demo link or video is visible.
- Eval cases are documented.
- Limitations are honest.
