# Compliance Checklist

## Input

- [ ] 已标 L0–L3
- [ ] 工具 / 租户获批
- [ ] 已移除密钥和个人信息
- [ ] Source Registry 权限正确
- [ ] 使用范围符合数据 Owner 授权

## Tools

- [ ] MCP 在白名单
- [ ] 默认只读
- [ ] 网络域名受限
- [ ] 写操作需二次确认
- [ ] 日志和 Owner 明确
- [ ] 自动执行策略符合等级

## Output

- [ ] 无敏感信息泄露
- [ ] 事实有来源
- [ ] 生成 / mock 明确标识
- [ ] Diff、测试和回滚已审
- [ ] 对外发布已审批

## Close

- [ ] 临时数据已删除
- [ ] 决策已记录
- [ ] 失败样本已写回
- [ ] 下次复核日期已设置
