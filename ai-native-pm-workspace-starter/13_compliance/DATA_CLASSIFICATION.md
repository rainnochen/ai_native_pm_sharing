# Data Classification

> 以下为工作模板，不替代组织正式制度。

| Level | Definition | External tools | Internal tools | Human Gate |
|---|---|---|---|---|
| L0 | 公开资料、公开代码、合成数据 | 可按组织政策 | 可 | 发布审查 |
| L1 | 内部非敏感 | 脱敏摘要 + 组织政策确认 | 优先 | Owner |
| L2 | 业务敏感、待发布方案、内部指标 | 批准环境 + 最小必要摘要 | 批准环境 | 业务 / 数据 / 合规 |
| L3 | 代码、用户数据、商业数据、密钥、高敏信息 | 授权隔离环境 | 授权隔离环境 | 强审批和审计 |
