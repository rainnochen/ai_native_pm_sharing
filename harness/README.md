# Harness

An AI Harness is the work environment around an AI model: context, skills, tools, rules, state, outputs, and evaluation.

PM Harness design should answer:

- What context can the AI see?
- What skills can it run?
- What tools can it use?
- What must it never do automatically?
- What output contract must it satisfy?
- How will results be evaluated?
- Where will failures and decisions be recorded?
