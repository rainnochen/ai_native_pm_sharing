# Claude Guidance

When using Claude or Claude Code with this repository:

1. Read `AGENTS.md` first.
2. Do not invent sources, metrics, or company claims.
3. Keep PM templates copyable and practical.
4. Ask before using sensitive data or external actions.
5. When editing slides, keep `ai-native-pm-sharing-draft.md` and `ai-native-pm-sharing.html` aligned.
6. Prefer small, reviewable changes.

Recommended prompt:

```text
Read AGENTS.md and the relevant template. Improve this PM workflow while preserving Purpose, Required Inputs, Procedure, Output Contract, Quality Checks, Human Gates, Example, and Failure Modes. Do not add unsafe automation.
```
